# PROJETO E-COMMERCE
API RESTful Back-End desenvolvida em C++ para um desafio de e-commerce. O projeto implementa um CRUD completo em 3 entidades relacionadas (Cliente, Produto, Compra) utilizando MySQL containerizado via Docker. Inclui um sistema de autenticação JWT com controle de acesso diferenciado (Usuário Padrão e Administrador) e lógica transacional de compra com redução de estoque.
